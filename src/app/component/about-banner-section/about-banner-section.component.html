CARGO TRACKING NOTE (CTN)

The CTN form (commonly,one form per BL) mainly include:

  • IMPRTER,EXPORTER & FORWARDER(S) DATA : Name,Address,Phone No,e-mail etc.
  • GOODS : a Draft copy can be used for the draft
  • CHARGES : Incoterm, reference, and the date have to be mentioned.
  • DOCUMENTS It has to be provided if the freight value is not mentioned on the commercial invoice
